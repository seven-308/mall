# 卡片群组展示页面 Demo

## 📄 文件说明

- **card-group-demo.html** - 根据 Linear issue ZLT-5 截图实现的前端页面

## 🎨 功能特性

### 1. 页面布局
- ✅ 响应式设计，支持移动端和桌面端
- ✅ 现代化渐变背景
- ✅ 卡片式布局，阴影和动画效果

### 2. 顶部信息栏
- ✅ 场景标题显示："场景2：有卡片无项目"
- ✅ iCafe图标和卡片数量统计
- ✅ 刷新按钮（带动画效果）
- ✅ 添加卡片按钮
- ✅ 分页导航（前一张/后一张）
- ✅ 页码显示（1/4）

### 3. 卡片展示
- ✅ 类型标签：Story/Task/Bug
- ✅ 状态标签：进行中/待处理/已完成（带颜色区分）
- ✅ 工单编号：WF-1234
- ✅ 卡片标题
- ✅ 卡片描述
- ✅ 操作按钮组：
  - 📋 复制按钮（支持一键复制）
  - 🔗 分享按钮
  - 🔗 关联项目按钮
  - ⋮ 更多操作按钮

### 4. 交互功能
- ✅ 卡片翻页（点击前后按钮或使用键盘左右箭头）
- ✅ 刷新功能（Ctrl+R 快捷键）
- ✅ 复制功能（一键复制卡片信息到剪贴板）
- ✅ 按钮悬停效果（颜色变化和上浮动画）
- ✅ 通知提示（右上角弹出提示）
- ✅ 响应式布局（移动端自适应）

### 5. 内置数据
页面包含 4 张示例卡片：
1. 用户登录功能优化 (Story - 进行中)
2. 商品详情页重构 (Task - 待处理)
3. 修复购物车数量异常 (Bug - 已完成)
4. 订单管理系统优化 (Story - 进行中)

## 🚀 使用方法

### 方法一：直接打开
1. 在浏览器中打开 `card-group-demo.html` 文件
2. 无需服务器，可直接运行

### 方法二：使用本地服务器
```bash
# 使用 Python
cd /workspace
python3 -m http.server 8000

# 使用 Node.js
npx http-server

# 然后访问: http://localhost:8000/card-group-demo.html
```

## ⌨️ 键盘快捷键

- `←` 左箭头：上一张卡片
- `→` 右箭头：下一张卡片
- `Ctrl + R`：刷新卡片列表
